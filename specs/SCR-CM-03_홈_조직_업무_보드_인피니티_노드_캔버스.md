## 2. 레이아웃

상단 탭(인사·자금·관리·기술·정보) + 도구 팔레트(도구·회의록·규정·업무평가·리스크평가) + 기간 토글(주·월·분기) + 인피니티 노드 캔버스(조직 1~4depth→조직원→업무 순차 전개) + 우측 DRAWER(목표관리·BI)

**Drawer '결재 프로세스' 미니 섹션(REF-22 · 목표·BI 위에 추가)**: ① AI 요약 1줄(`✦`) → ② 결재선 미니 스텝 `기안 ● — 검토 ◐(현재) — 승인 ○` (현재 단계 warn 노랑·지연/반려 risk 빨강) → ③ 담당·소요일·전결/대결 뱃지 → ④ CTA `[결재 상세 보기]`. 업무 노드·워크플로우 카드에는 **결재상태 칩**(회색 기안/노랑 결재중/초록 완료/빨강 반려)을 표시하고, 조직·과제 노드는 하위 최악 심각도로 **롤업**(빨강>노랑>초록>회색). Drawer 폭 확대(개선요구 "목표관리 BI 2배")와 함께 적용.

```
┌────────────────────────────────────┐
│ 헤더(GNB·AI질문창)                │
├────────────────────────────────────┤
│ 필터/탭/팔레트 바                    │
├────────────────────────────────────┬──────┤
│ 본문                               │ 우측 │
│                                    │Drawer│
└────────────────────────────────────┴──────
```

**주요 컴포넌트**: TopTabs,ToolPalette,InfiniteCanvas(React Flow 커스텀),OrgNode,MemberNode,TaskNode,ToolNode,Edge,Drawer,SegToggle,Minimap + **ApprovalBadge(노드 칩),ApprovalMiniSteps,AiSummaryLine** (REF-22 신규 · 색은 `--color-*` 토큰만)
