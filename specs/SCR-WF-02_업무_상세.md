## 2. 레이아웃

우측 Drawer · 탭(업무/하위업무/마일스톤/담당자&참여자/댓글 + 프로세싱 뷰/프로세스 차트(간트)/규정 연계)

**워크플로우 탭 구성(REF-22)**: ① 상단 **AI 요약 배너**(1–2줄·§2 전역 규약) → ② 업무 단계(STEP: 접수→검토→추진) 진행바 → ③ **결재선 타임라인/스테퍼(가로)** — `기안(DRAFT)→검토(LEADER)→승인(OWNER)`, 각 노드에 담당자·직급·상태색·소요일·기한·전결/대결 뱃지, 평균 초과 단계에 빨강 "+n일" 뱃지 → ④ 하단 CTA `[BI로 원인 분석]` `[보고서 작성]`. 결재는 각 업무 단계의 게이트로 표현.

```
┌────────────────────────────────────┐
│ 헤더(GNB·AI질문창)                │
├────────────────────────────────────┤
│ 필터/탭/팔레트 바                    │
├────────────────────────────────────┬──────┤
│ 본문                               │ 우측 │
│                                    │Drawer│
└────────────────────────────────────┴──────
```

**주요 컴포넌트**: Drawer,Tabs,ProgressBar,StateFlow,Flow,Gantt,List,Button + **AiSummaryBanner,ApprovalTimeline(스테퍼),DelayBadge,CtaBar** (REF-22 신규 · 색은 `--color-*` 토큰만)
